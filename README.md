# Stock Market Analysis

##  Description
Projet d'analyse exploratoire des marchés boursiers (S&P 500) 
visant à identifier des tendances, corrélations et insights 
financiers à partir de données historiques.

##  Outils & Technologies
- **R** — nettoyage et analyse statistique
- **SQL** — extraction et manipulation des données
- **Matplotlib / Seaborn** — visualisations

##  Structure du projet
```
stock-market-analysis/
│
├── README.md          
├── data/              
│   ├── raw/           
│   └── cleaned/       
├── sql/               
├── notebooks/         
├── visualizations/    
└── reports/           
```

## Objectifs de l'analyse
- Analyser l'évolution des prix historiques du S&P 500
- Identifier les secteurs les plus performants
- Visualiser les tendances et volumes d'échange
- Dégager des insights économiques clés

## Dataset
- **Source** : [Kaggle - Stock Market Data](https://www.kaggle.com/datasets/paultimothymooney/stock-market-data)
- **Contenu** : Prix historiques, volumes, indices boursiers
